# ShiftinBits Homebrew Tap

Homebrew formulae and casks for [ShiftinBits](https://github.com/ShiftinBits) projects.

## Install

```bash
# Add the tap (one time)
brew tap shiftinbits/tap

# Then install any package
brew install pmux
```

Or install directly without tapping first:

```bash
brew install shiftinbits/tap/pmux
```

## Available Packages

| Package | Description |
|---------|-------------|
| [`pmux`](https://github.com/ShiftinBits/pmux-agent) | The [Pocketmux](https://pmux.io) agent — a transparent drop-in `tmux` replacement that makes your terminal sessions accessible from your phone over an encrypted peer-to-peer connection |

## Updating

```bash
brew update
brew upgrade pmux
```
